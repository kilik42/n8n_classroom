# n8n in GitHub Codespaces

This template gives you your own private n8n automation environment with **zero installs**.

##  Start Here

1. Fork this repository  
2. Click **Code → Create Codespace on Main**  
3. Wait 1–2 minutes while the container builds  
4. A browser tab will open automatically with n8n  
5. Start building workflows!

## What You Can Do

- Build automations  
- Call APIs  
- Use AI nodes  
- Create webhooks  
- Transform data  
- Connect services  

Your work is saved automatically inside the Codespace.

##  When You're Done

Just close the Codespace.  
You can reopen it anytime and your n8n instance will still be there.
